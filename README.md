# BeautyStore - HTML & CSS Website Project

A clean and well-structured beauty e-commerce website inspired by Nykaa, built using only HTML and CSS.

## Project Overview

This project demonstrates fundamental web development skills including page structuring, navigation, layout design using Flexbox/Grid, and responsive styling. The website contains 8 pages with consistent navigation and footer across all pages.

## Pages Included

1. **Home Page (index.html)** - Hero banner with featured products
2. **Products Page (products.html)** - Grid layout of all products
3. **Product Details Page (product1.html)** - Single product view with full details
4. **Cart Page (cart.html)** - Shopping cart with table layout
5. **Checkout Page (checkout.html)** - Checkout form with customer details
6. **About Us Page (about.html)** - Company information and team members
7. **Contact Page (contact.html)** - Contact form and information
8. **Categories Page (categories.html)** - Product categories display

## Features

- **Clean & Professional Design** - Nykaa-inspired pink color scheme (#fc2779)
- **Responsive Layout** - Uses CSS Flexbox and Grid
- **Consistent Navigation** - Sticky navigation bar across all pages
- **Unified Footer** - Contact information and social links on every page
- **No JavaScript** - Pure HTML & CSS implementation
- **Placeholder Images** - Uses Unsplash CDN images
- **Form Validation** - HTML5 form validation (basic)
- **Hover Effects** - Smooth transitions on cards and buttons

## How to Run

1. Download all files to a folder
2. Open `index.html` in any web browser
3. Navigate through pages using the navigation menu
